# Task Manager App (Laravel)

Simple CRUD-based Task Management App built with Laravel 10 + MySQL.

## Features
- Add, edit, delete tasks
- Mark task as done/undone
- Due date tracking

## Tech Stack
- Laravel 10
- PHP 8.1
- MySQL
- Blade templates
- Tailwind CSS

## Setup
1. Clone repo
2. `composer install`
3. Create `.env` and set DB credentials
4. `php artisan migrate`
5. `php artisan serve`
